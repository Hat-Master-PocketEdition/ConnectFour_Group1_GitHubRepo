CIS 153 Final Project: Connect Four Program
Created by Aminah Bakr, Austin Watkins, David Sargent
=======================================================
This is a program for the game Connect Four.
Upon initialization User is presented with options of navigation:
The Start form has the User greeted by the game type. Connect Four! It has the choices of 1 Player, 2 Player, and finally Stats.
1 Player puts the User against the CPU in which they battle in the board until one can produce a chain of four. Or the board fills and it is a tie.
2 Player allows two Users (locally) to play against each other in the rules, however the game is not documented, so paper is required (nor provided).
The Stats page house the data from User's many games against the CPU, showing Game Count and percentages to User Wins, and CPU Wins. With Win Count and ties for the User, and wins for the CPU.
=======================================================
Color Assignment for 1 Player - {Red - User; Yellow - CPU}
Color Assignmnet for 2 Player - {Red - Player 1; Yellow - Player 2}
The game is to get 4 chips in a row, either horizontally, vertically, and diagonally.
Once the game is completed, the user(s) are met with a review board to see the board state and see who won and where!
Run another or check your stats, it's up to you!
